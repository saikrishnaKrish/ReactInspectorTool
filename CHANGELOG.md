# Changelog

## v2.7.0 — 2026-03-03
- **Major MUI Enhancement**: Intelligent MUI component detection (100+ MUI components)
- **Smart Hierarchy Display**: Full component tree with visual MUI/App component distinction (📦 badge)
- **Framework Component Filtering**: Automatically filters React Router, Redux, Apollo, and other framework components
- **Compound Component Support**: Properly detects and filters CardContent, ListItemText, TableCell, AccordionSummary, etc.
- **Context Filtering**: Filters MUI contexts (GridContext, CardContext, ModalContext, etc.) from hierarchy
- **Pattern-Based Detection**: Detects MUI components via 8 intelligent signals (props, naming patterns, parent chains, prototypes)
- **Production-Grade Architecture**: Scales automatically without hardcoding component lists
- **Improved UI/UX**: Cleaner hierarchy display showing only meaningful app components
