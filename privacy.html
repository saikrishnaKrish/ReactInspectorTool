Privacy Policy – React Inspector Pro

Effective Date: January 2026

React Inspector Pro does not collect, store, transmit, or sell any personal user data. The extension operates entirely locally in the browser.

Data Usage

  • No personal information collected
  • No browsing history collected
  • No authentication or credentials
  • No analytics or tracking
  • No network requests

Storage

User preferences such as theme and layout are stored locally using Chrome’s extension storage APIs and never leave the device.

Permissions

  • activeTab – inspect current page on user action
  • storage – save preferences locally
  • clipboardWrite – copy inspected data on user action
  • <all_urls> – inspect React components on any site
