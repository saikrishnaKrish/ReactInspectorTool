# Privacy Policy - React Inspector Pro

**Effective Date:** January 2026

## Overview

React Inspector Pro ("the Extension") is committed to protecting your privacy. This Privacy Policy explains our data practices for the Chrome extension.

## Data Collection & Storage

### What We DO Collect:
- **Local Storage Only**: Theme preference (dark/light mode) stored locally in your browser
- **Local Storage Only**: Recent color picks from the eyedropper tool stored in your browser
- **Local Storage Only**: Panel collapse/expand state preferences

### What We DO NOT Collect:
- No personal information
- No browsing history
- No credentials or passwords
- No analytics or tracking
- No cookies
- No server communications (fully offline extension)
- No user profiling

## Permissions Explanation

The Extension requires the following permissions to function:

| Permission | Purpose |
|-----------|---------|
| `<all_urls>` | Allows inspection of React components on any website you visit |
| `storage` | Stores your theme preference and color history locally |
| `activeTab` | Enables Alt+I shortcut to toggle the inspector |
| `clipboardWrite` | Allows copying component data, code, and styles to clipboard |

## Data Handling

- **All data is local**: Everything is stored in `localStorage` within your browser
- **No transmission**: Your data is never sent to any server
- **No third parties**: We don't share data with anyone
- **Fully offline**: The extension works completely without internet connectivity

## User Rights

You have full control:
- Clear your browser's local storage to delete all stored preferences
- Uninstall the extension at any time to remove all associated data
- The extension cannot access data outside your active browser

## Security

- No external network requests
- No remote code loading
- No tracking pixels or beacons
- All code is open-source and inspectable

## Changes to Privacy Policy

We may update this policy. Changes will be reflected in the CHANGELOG with new version releases.

## Questions or Concerns?

For inquiries about this Privacy Policy, please contact us via the extension's GitHub repository.
